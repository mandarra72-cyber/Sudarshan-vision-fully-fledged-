# Sudarshan English Degree College

Official website for Sudarshan English Degree College.

## Description

A modern, comprehensive web application serving as the official digital platform for Sudarshan English Degree College. This responsive website provides prospective students, current students, and visitors with seamless access to essential college information and interactive features designed to enhance the admissions experience.

The platform offers detailed information about undergraduate programs including BCA (Bachelor of Computer Applications), BCom (Bachelor of Commerce), and BA (Bachelor of Arts), complete with course descriptions, eligibility criteria, and career prospects. Students can explore program details through an intuitive interface designed for easy navigation and accessibility.

At the heart of the website is a sophisticated online application system featuring real-time form validation, ensuring accurate data submission while providing instant feedback to applicants. The integrated campus visit scheduler allows prospective students and parents to book personalized tours by selecting preferred dates and time slots, making the college exploration process convenient and organized.

Communication is streamlined through a professional contact form that enables direct inquiries about admissions, programs, facilities, and general information. Users can also subscribe to the college newsletter to stay informed about events, announcements, admission updates, and student achievements.

A standout feature is the automated email notification system powered by EmailJS, which ensures users receive immediate, professional confirmations for every interaction. Whether submitting an application, scheduling a campus visit, sending a contact inquiry, or subscribing to the newsletter, users receive detailed confirmation emails with relevant information and next steps.

Built with cutting-edge technologies including React with TypeScript for robust development, Tailwind CSS for modern responsive design, and a Node.js/Express backend with MongoDB database integration, the website delivers optimal performance across all devices. The platform emphasizes user experience through smooth animations, mobile-first design, and accessibility compliance.

This digital solution transforms the traditional admissions process by providing 24/7 access to college information, automating communication workflows, and creating an efficient, transparent pathway for students to connect with Sudarshan English Degree College.

## Setup

```bash
npm install
cd server && npm install
cd ..
npm run dev
```

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Node.js + Express
- MongoDB

## Features

- **Program Information**: Browse undergraduate programs (BCA, BCom, BA) with detailed course information
- **Online Applications**: Submit applications online with real-time form validation
- **Campus Visit Booking**: Schedule campus tours with preferred date and time
- **Contact Forms**: Get in touch with admissions team
- **Newsletter Subscription**: Stay updated with college news and events
- **Email Automation**: Automated confirmation emails for all user interactions using EmailJS
  - Application confirmation emails
  - Campus visit confirmation emails
  - Contact form acknowledgment emails
  - Newsletter welcome emails
